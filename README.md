\# DeepSeek Research Outreach Agent



This project generates structured 3-sentence academic outreach messages using DeepSeek API.



\## Features

\- Strict 3-sentence format

\- Academic tone

\- Personalized output



\## Setup



1\. Install dependencies:

pip install -r requirements.txt



2\. Add API key:

Create .env file and add:

DEEPSEEK\_API\_KEY=sk-d6aaf104a9104ca4a19553b1a304eb9c



3\. Run:

python agent\_outreach.py
